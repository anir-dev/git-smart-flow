## Troubleshooting First

Run the built-in diagnostic tool:

```bash
git-smart-flow doctor
```

This checks your Git installation, Node.js version, config files, AI provider availability, and hook setup.

## Getting Help

**For questions and discussions:**  
Use [GitHub Discussions](https://github.com/anir-dev/git-smart-flow/discussions). This is the best place for:
- "How do I...?" questions
- Feature ideas and suggestions
- Sharing your setup

**For bug reports:**  
Open a [GitHub Issue](https://github.com/anir-dev/git-smart-flow/issues/new?template=bug_report.md) using the bug report template.

**For security vulnerabilities:**  
Do **not** open a public issue. Instead:
- Use [GitHub Private Security Advisories](https://github.com/anir-dev/git-smart-flow/security/advisories/new)
- Or email `anir.dev.corp@gmail.com`

See [SECURITY.md](SECURITY.md) for details.

## Response Times

This is an open source project maintained in spare time. Responses are best-effort:

| Type | Target |

| Feature requests | No guarantee |
